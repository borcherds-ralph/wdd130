Starting HTML syntax

This is the basic HTML format that you should be using for correct HTML5 standards.

The main tag below has the various section type tags that are allowed to be used in the main and footer sections. Make sure to read the Welcome to Week 10 Announcement for a chart that shows when each should be used.

The heading tags can be used in any area. Just remember that h1 should only be used once per page. There is one exception and that is when it is inside the article tag.

Correct HTML format that must be used.  Comments are bold and in green.

 

<!DOCTYPE html>

        <html lang="en">

        <head>

            <meta charset="UTF-8">

            <meta http-equiv="X-UA-Compatible" content="IE=edge">

            <meta name="viewport" content="width=device-width, initial-scale=1.0">

            <title>Page Title</title>

            <link rel="stylesheet" href="styles/style.css">   <!-- This must go here after the other tags -->
    <!-- Do not link to Font-Awesome for school work. That does not validate. 
        Copy SVG code from https://heroicons.com for icons in your page if you want to use them -->
</head>

        
<body>

            <header>

                <img src="images/logo.svg" alt="Company Logo" class="logo">   <!-- This is for the logo only -->
        <nav>

                    <!-- Links to the 3 website pages go in this area -->
        </nav>

            </header>

            <main>

                <!-- The ALL content between the header and footer goes in here These are valid sub-sections
        inside the main area  Hero images should go here as well -->
        <section></section>

                <div></div>

                <article></article>

                <aside></aside>

                <!-- You can use all other formatting tags inside the main or the other sections listed
        h1, h2, h3, h4, h5,h6
        img
        p
        table
        etc....
        -->
    </main>

            <footer>

                <!-- 
        Link to site plan must go here only 
        Copyright notice must go here
        This is where you will put any links to these legal documents: terms of use, disclaimer, and Privacy policy 
        -->
    </footer>

        
    <!-- JavaScript goes here 99% of the time. The only other place in the head and only when required -->
    <!-- Modals code go here -->
</body>

        
</html>
