Folder and File structure

This is what you should have by week 4. 

 

wdd130 root folder should have these folders and file.  I have also listed which weeks each folder or file is supposed to be created. 

aboutme/     <- Week 1
practice/    <- Week 2
wwr/         <- Week 3

 

The wwr folder should have these folders and files

images/                  <- Week 3
styles/                  <- Week 2
index.html               <- Week 3 assignment
site-plan-rafting.html   <- Week 2

 

The aboutme folder should contain the index.html file for your week 1 Getting to Know You discussion board post.

 

wwr/images folder should have all the images for the rafting site homepage For example this list.

wwr/styles folder

wdd130/wwr/styles/site-plan-rafting.css    <- Week 2
wdd130/wwr/styles/style.css                <- Week 4

 

The practice folder will contain any practice files you have crate or challenge assignments that are not submitted for grading.

 

There will be a lot of small details for you to do. Pay close attention to the details.
